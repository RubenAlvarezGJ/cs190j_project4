# Player Character
## idle

16-bit SNES-era pixel art, top-down overhead 3/4 perspective in the visual
lineage of A Link to the Past. Hand-pixeled look, clean dark outlines,
limited palette, no anti-aliasing, nearest-neighbor crisp edges.

SUBJECT: a lone post-apocalyptic survivor — the player character of a
zombie survival game. Pre-shrine "scavenger" look: rugged improvised
clothing (worn jacket, wrapped/taped boots, fingerless gloves, a cloth or
scarf), no armor yet. Lean, capable but vulnerable. Holding a simple
pickaxe in one hand. Character should read clearly from above.

CHARACTER SHEET: the SAME character shown in 4 separate idle poses —
facing DOWN (toward camera), facing UP (back to camera), facing RIGHT,
and facing LEFT — arranged in a single row on a transparent background.
Consistent proportions and palette across all four. Roughly 24x24 px per
character cell, scaled up cleanly for visibility.

PALETTE + MOOD: desaturated and muted — rust browns, ash grays, faded
military green, weathered cloth, dead-grass yellow. Gritty and worn, but
the figure should still be slightly warmer/lighter than a typical ruined
background so it stays readable. Tense survival tone, NOT gory or horror —
T-for-Teen, stylized.

FORMAT: pixel-art sprite reference sheet, transparent background,
limited palette, sharp pixels, no blur, no smoothing, no gradients.

CONSISTENCY: this is the master reference for the entire "fLoW" art set;
all later sprites (zombies, armored player, props) must match this exact
style, perspective, and palette.
